# MyDefrag Script Extension

Language support for MyDefrag (.MyD/.MyDc).

## Features

### Editing
- Syntax highlighting
- Intelligent snippets
- Code navigation

### Validation
- Custom MyDefrag language server.
- Real-time parsing and validation while editing.
- Full-script detection for `.MyD` files.
- Fragment-aware parsing for `.MyDc` files.
- Workspace-wide validation and diagnostics.
- Tunable diagnostic severity by message category (missing files, paths, parent hints, etc.).

### Navigation
- Go to Definition
- Find References
- Include navigation
- Preview
