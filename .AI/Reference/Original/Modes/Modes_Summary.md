# Models Summary

| Mode                | Purpose                                                                                             | Typical Use                                                                |
| ---------------------| -----------------------------------------------------------------------------------------------------| ----------------------------------------------------------------------------|
| **Minimal**         | Fast startup with only essential context.                                                           | Small edits, questions, simple tasks.                                      |
| **Standard**        | Default mode. Loads the role, required procedures, required standards, and relevant context.        | Most development work.                                                     |
| **Detailed**        | Extended planning, analysis, diagnostics, architecture, code reviews, or when explicitly requested. | Reviews, major design work, debugging, documentation generation.           |
| **Verbose (Audit)** | Standard or Detailed mode plus comprehensive execution evidence and reporting.                      | Reviews, benchmarking, MCP development, regression testing, AI evaluation. |
