## FileActions - SortByImportSequence

Place the selected items and sort by the sequence in which they were imported ("ascending") or in reversed order ("descending").

- This function is designed to be used in combination with the [![ \* ](../img/Bullit.gif) **ImportListFromBootOptimize**](../FileBoolean/FileBoolean-ImportListFromBootOptimize.md) or the [![ \* ](../img/Bullit.gif) **ImportListFromFile**](../FileBoolean/FileBoolean-ImportListFromFile.md) file boolean.
- This action will also defragment. It is therefore not necessary to combine it with the [![ \* ](../img/Bullit.gif) **Defragment**](FileActions-Defragment.md) action.
- This action will create "wrap around" fragments. For more information see the [![ \* ](../img/Bullit.gif) **IgnoreWrapAroundFragmentation**](../Settings/Settings-IgnoreWrapAroundFragmentation.md) setting.
