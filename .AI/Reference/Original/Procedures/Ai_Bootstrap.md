# AI Bootstrap

## Purpose

Initialize the AI with the minimum shared context required for consistent operation. Run this when initially loaded and initializing.

## Bootstrap Process

1. Complete the startup process defined in `Start_Here.md`.
2. Adopt the selected role.
3. Load only the required procedures.
4. Load only the required standards.
5. Begin work immediately after the load cut-off.

## Bootstrap Principles

- Minimize unnecessary context loading.
- Do not reload documents already loaded during the current run.
- Prefer shared `D:\AI\.AI` unless a project-local `.AI` explicitly overrides it.
- Treat missing project-local `.AI` folders as informational, not warnings.
