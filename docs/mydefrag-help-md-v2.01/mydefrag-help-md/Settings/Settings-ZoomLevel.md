## Settings - ZoomLevel

Set the zoom level of the MyDefrag diskmap display.

- If NUMBER is not a number in the zoomlevel pull-down menu, then the menu will not show a selected zoomlevel.
- Setting the zoomlevel works best after the first [![ \* ](../img/Bullit.gif) **VolumeSelect**](../Scripts/Scripts-VolumeSelect.md) or after [![ \* ](../img/Bullit.gif) **WindowSize**](Settings-WindowSize.md).

### Syntax

|                                                         |
|---------------------------------------------------------|
| `ZoomLevel(`[`NUMBER`](../Scripts/Scripts-NUMBER.md)`)` |

### Parameters

|                                                                                      |
|--------------------------------------------------------------------------------------|
| The zoom level, a positive integer number greater than zero. Default zoomlevel is 1. |

### Example

|                   |
|-------------------|
| `ZoomLevel(1024)` |
