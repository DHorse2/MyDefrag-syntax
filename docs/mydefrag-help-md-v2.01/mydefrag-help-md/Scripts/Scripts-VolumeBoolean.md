## Scripts - VolumeBoolean

Select volumes for processing. There are several functions to choose from, and they can be combined in expressions with AND, OR, NOT, and parenthesis.

- Volumes that are selected with the VolumeBoolean of the global [![ \* ](../img/Bullit.gif) **ExcludeVolumes**](../Settings/Settings-ExcludeVolumes.md) setting are automatically excluded in the VolumeBoolean of [![ \* ](../img/Bullit.gif) **VolumeSelect**](Scripts-VolumeSelect.md) statements.
