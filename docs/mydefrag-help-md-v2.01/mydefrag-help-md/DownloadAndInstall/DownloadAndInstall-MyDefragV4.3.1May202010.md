## Download and install - MyDefrag v4.3.1 (May 20, 2010)

Changes:

- Added the Windows "bootstat.dat" file to the build-in list of special exceptions.
- To my great chagrin I have found that some programs can crash (most notably MySQL and uTorrent) if the disk on which they have open files is dismounted with the MyDefrag [![ \* ](../img/Bullit.gif) **DismountVolume**](../VolumeActions/VolumeActions-DismountVolume.md) command. The command was used in the DataDisk scripts that were introduced in the previous version. I have now removed it.
