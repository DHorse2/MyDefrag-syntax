## FAQ Download And Install - The screensaver does not work?

On Vista the screensaver only works with User Account Control turned off (see Control Panel, User Accounts, Turn User Account Control on or off). The problem is because of general Windows security, complain to Microsoft, not me.

If UAC is not the problem then see the MyDefrag "Settings.MyD" file and turn on one of the debug settings. The screensaver will then create a logfile called "c:\Program Files\MyDefrag v4.3.1\MyDefrag.debuglog" to help you solve the problem.
