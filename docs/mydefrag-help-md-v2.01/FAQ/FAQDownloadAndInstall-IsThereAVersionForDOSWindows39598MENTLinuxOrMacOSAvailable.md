## FAQ Download And Install - Is there a version for DOS, Windows 3/95/98/ME/NT, Linux, or Mac OS available?

No. MyDefrag is based on the Microsoft defragmentation library, which is only available for the Windows versions after Windows 2000, not for DOS, Windows 3.\*, Windows 95/98/ME, and not for other operating systems such as Linux and Mac OS. The defragmentation library is available on Windows NT, but NT lacks certain other libraries that MyDefrag requires.

- MS-DOS 6.x and Windows 9x-systems come with a built-in defragmentation utility called Defrag.
- For Windows 95/98/Me see the [![ \* ](img/Bullit.gif)](http://www.vopt.com/) [**Vopt**](http://www.vopt.com/) defragger.
