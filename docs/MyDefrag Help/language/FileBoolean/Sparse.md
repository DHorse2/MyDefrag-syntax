# FileBoolean - Sparse

Select all the items that are sparse (yes) or not sparse (no).

### Syntax

```mydfrg
Sparse(yes)
Sparse(no)
```

### Example

```mydfrg
FileSelect
  # Select all the items that are sparse.
  Sparse(yes)
FileActions
  ....
FileEnd
```
