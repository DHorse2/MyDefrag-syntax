## FAQ Using - Can I "hide" the program from users?

Add a line with the [![ \* ](img/Bullit.gif) **WindowSize**](../Settings/Settings-WindowSize.md) setting to your "c:\Program Files\MyDefrag v4.3.1\Settings.MyD" script.

### Example

|                         |
|-------------------------|
| `WindowSize(invisible)` |

  
Note: I advise against this, I think it will needlessly worry the user when he sees a lot of mysterious disk activity.
