# Review Checklist

Use the following sections in every report.

1.  Executive Summary
2.  Architecture Review
3.  Parsing System Review
4.  Language Server Review
5.  Diagnostics Review
6.  Extension Review
7.  Logging System
8.  Configuration System
9.  Performance Review
10. Code Quality
11. Security Review
12. Reliability Review
13. Technical Debt
14. Refactoring Opportunities
15. Future Features
16. Positive Design Decisions
17. Overall Assessment

For the Overall Assessment include 1--10 ratings for:

-   Architecture
-   Code Quality
-   Maintainability
-   Parser
-   Language Server
-   Extensibility
-   Documentation
-   Production Readiness

Finish with the **Top 25 engineering improvements ranked by value**.

Do not generate code or patches unless specifically requested.
