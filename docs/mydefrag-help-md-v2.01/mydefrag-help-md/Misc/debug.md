## MyDefragDebugVersion - MyDefrag debug version

This page is for people that have received a "Please download and install the debug version of MyDefrag" message in the MyDefrag debug logfile. Please do not download and install the debug version unless you have received that message.

Release date: May 21, 2010  

[TABLE]

- Uninstall MyDefrag.
- Download the debug version from the above link.
- Install the debug version.
- Change the "Debug()" setting in the Settings.MyD file. Usually the "Debug(175)" setting will generate enough detail.
- Run MyDefrag as before.
- When and if MyDefrag crashes please post the MyDefrag.debuglog on the forum. Do not post the complete file, but only the first and last few hundred lines, enough to demonstrate the problem. Also post any other information that you feel might be useful.
- Thanks for your help, I appreciate it!
