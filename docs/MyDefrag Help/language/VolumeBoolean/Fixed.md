# VolumeBoolean - Fixed

Select the volume if it has Fixed(yes) or not Fixed(no) media, for example,
  a hard drive or flash drive.

### Syntax

```mydfrg
Fixed(yes)
Fixed(no)
```

### Example

```mydfrg
VolumeSelect
  Fixed(yes)
VolumeActions
  ...
VolumeEnd
```
