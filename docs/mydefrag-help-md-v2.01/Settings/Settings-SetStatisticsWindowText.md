## Settings - SetStatisticsWindowText

Set the text that is displayed in the statistics window. The text can be changed as many times as you want, for example a text while analyzing a disk, while defragmenting, and when finished. See the [![ \* ](../img/Bullit.gif) **Variables**](../Scripts/Scripts-Variables.md) for a list of variables that can be used in the text.

### Syntax

|                                                                       |
|-----------------------------------------------------------------------|
| `SetStatisticsWindowText(`[`STRING`](../Scripts/Scripts-STRING.md)`)` |
