## Scripts - VolumeActions

The VolumeActions keyword is part of the [![ \* ](../img/Bullit.gif) **VolumeSelect**](Scripts-VolumeSelect.md) structure and specifies the actions to be done on the selected volumes. There are several actions to choose from, but the basic workhorse is the [![ \* ](../img/Bullit.gif) **FileSelect**](../VolumeActions/VolumeActions-FileSelect.md) structure.
