# FileBoolean - Readonly

Select all the items that are read-only (yes) or not read-only (no).

### Syntax

```mydfrg
Readonly(yes)
Readonly(no)
```

### Example

```mydfrg
FileSelect
  # Select all the items that are read-only.
  Readonly(yes)
FileActions
  ....
FileEnd
```
