## Scripts - FileBoolean

Select items (files, directories) for processing. There are several functions to choose from, and they can be combined in expressions with AND, OR, NOT, and parenthesis.

- Files that are selected with the FileBoolean of the global [![ \* ](../img/Bullit.gif) **ExcludeFiles**](../Settings/Settings-ExcludeFiles.md) setting are automatically excluded in the FileBoolean of [![ \* ](../img/Bullit.gif) **FileSelect**](Scripts-FileSelect.md) statements.
