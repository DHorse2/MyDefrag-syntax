# VolumeBoolean - Cdrom

Select the volume if it is a Cdrom(yes) or if it is not a Cdrom(no).

### Syntax

```mydfrg
Cdrom(yes)
Cdrom(no)
```

### Example

```mydfrg
VolumeSelect
  Cdrom(yes)
VolumeActions
  ...
VolumeEnd
```
