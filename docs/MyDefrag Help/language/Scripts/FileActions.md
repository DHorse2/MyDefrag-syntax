# Scripts - FileActions

The FileActions keyword is part of the

[**FileSelect**](../VolumeActions/FileSelect.md)

structure
  and specifies the actions to be performed on the selected items (files, directories).
  There are several actions to choose from.

### Example

```mydfrg
FileSelect
  ....
FileActions
  Defragment()
FileEnd
```
