[![ \* ](../img/Bullit.gif) **MyFragmenter**](../SeeAlso/SeeAlso-MyFragmenter.md)  
MyFragmenter is a small commandline tool included with MyDefrag to fragment existing files, to generate new fragmented files with random data, or to list fragmentation information about files. It is useful only for people who are testing defragmentation programs.

## Other links

[![ \* ](../img/Bullit.gif)](http://en.wikipedia.org/wiki/Defragmentation) [**Defragmentation**](http://en.wikipedia.org/wiki/Defragmentation)  
Wikipedia page about defragmentation.

[![ \* ](../img/Bullit.gif)](http://www.microsoft.com/technet/sysinternals/FileAndDisk/PageDefrag.mspx) [**Pagedefrag**](http://www.microsoft.com/technet/sysinternals/FileAndDisk/PageDefrag.mspx) by Microsoft TechNet (formerly Sysinternals)  
Paging files and registry hives cannot be defragmented on a running system. This sweet little utility will automatically defragment those files when the system is booting. The counterpart of this utility is "Contig", a small defragger for use on a running system, but my defragger is far superior.

[![ \* ](../img/Bullit.gif)](http://www.hdtune.com/) [**HD Tune**](http://www.hdtune.com/)  
Measure your harddisk and discover how much faster it is at the beginning than at the end. Can also monitor the temperature of your harddisk and scan for surface errors.

[![ \* ](../img/Bullit.gif)](http://www.microsoft.com/downloads/details.aspx?FamilyID=9D467A69-57FF-4AE7-96EE-B18C4790CFFD&displaylang=en) [**Windows Server 2003 Resource Kit Tools**](http://www.microsoft.com/downloads/details.aspx?FamilyID=9D467A69-57FF-4AE7-96EE-B18C4790CFFD&displaylang=en)  
A free collection of tools from Microsoft. Amongst other things it contains a "sleep.exe" utility that could be used to make a little .bat file to delay-start MyDefrag when booting the computer.

[![ \* ](../img/Bullit.gif)](http://technet.microsoft.com/en-us/sysinternals/bb897553.aspx) [**PsExec**](http://technet.microsoft.com/en-us/sysinternals/bb897553.aspx) by Microsoft TechNet (formerly Sysinternals)  
Handy utility for network administrators that can be used on a central computer to upload to and run MyDefrag on remote computers.

[![ \* ](../img/Bullit.gif)](http://technet.microsoft.com/en-us/sysinternals/bb897443.aspx) [**SDelete**](http://technet.microsoft.com/en-us/sysinternals/bb897443.aspx) by Microsoft TechNet (formerly Sysinternals)  
A small free utility that (amongst other things) can be used to wipe free space on disk by writing zero's. This is useful in combination with optimization when making compressed disk images. SDelete can be called from a MyDefrag script with the [![ \* ](../img/Bullit.gif) **RunProgram**](../Settings/Settings-RunProgram.md) action.

[![ \* ](../img/Bullit.gif)](http://www.microsoft.com/technet/sysinternals/ProcessesAndThreads/Autoruns.mspx) [**AutoRuns**](http://www.microsoft.com/technet/sysinternals/ProcessesAndThreads/Autoruns.mspx) by Microsoft TechNet (formerly Sysinternals)  
Handy utility to discover all the things that are started when windows is booting.

[![ \* ](../img/Bullit.gif)](http://gparted.sourceforge.net/) [**Gnome Partition Editor (GParted)**](http://gparted.sourceforge.net/)  
For creating, deleting, resizing, moving, checking and copying partitions, and the file systems on them. This is useful for creating space for new operating systems (works with Vista system and data partitions), reorganizing disk usage, copying data residing on hard disks and mirroring one partition with another (disk imaging).

[![ \* ](../img/Bullit.gif)](http://en.wikipedia.org/wiki/Harddisk) [**Hard disk drive**](http://en.wikipedia.org/wiki/Harddisk)  
[![ \* ](../img/Bullit.gif)](http://en.wikibooks.org/wiki/Minimizing_hard_disk_drive_failure_and_data_loss#Temperature_control) [**Minimizing hard disk drive failure and data loss**](http://en.wikibooks.org/wiki/Minimizing_hard_disk_drive_failure_and_data_loss#Temperature_control)  
Some links to technical background information about harddisks.
