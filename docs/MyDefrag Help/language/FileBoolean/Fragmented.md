# FileBoolean - Fragmented

Select all the items that are fragmented(yes) or not fragmented(no).

### Syntax

```mydfrg
Fragmented(yes)
Fragmented(no)
```

### Example

```mydfrg
FileSelect
  # Select all the fragmented items.
  Fragmented(yes)
FileActions
  ....
FileEnd
```
