# VolumeBoolean - Remote

Select the volume if it is Remote(yes) or not Remote(no). A remote disk
  is usually a network disk.

### Syntax

```mydfrg
Remote(yes)
Remote(no)
```

### Example

```mydfrg
VolumeSelect
  Remote(yes)
VolumeActions
  ...
VolumeEnd
```
