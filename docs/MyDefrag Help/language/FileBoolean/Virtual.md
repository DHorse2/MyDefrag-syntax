# FileBoolean - Virtual

Select all the items that have the "virtual" attribute set (yes) or not set (no).

### Syntax

```mydfrg
Virtual(yes)
Virtual(no)
```

### Example

```mydfrg
FileSelect
  # Select all the items that have the "Virtual" attribute.
  Virtual(yes)
FileActions
  ....
FileEnd
```
