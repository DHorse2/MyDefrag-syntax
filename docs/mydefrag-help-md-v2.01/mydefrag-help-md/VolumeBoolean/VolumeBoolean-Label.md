## VolumeBoolean - Label

Select the volume if it's label matches the STRING. The string may contain wildcard characters "\*" (zero or more characters) or "?" (a single character).

- The label of a volume is a small text that you have attached to the volume. The drive-letter is not a label, but the name of a volume, see the [![ \* ](../img/Bullit.gif) **Name**](VolumeBoolean-Name.md) volumeboolean.
- Unmounted volumes (for example floppies and cdroms) do not have a label.
